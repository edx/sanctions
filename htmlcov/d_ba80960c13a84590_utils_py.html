46def transliterate_text(text): 

47 """ 

48 Transliterate unicode characters into ASCII (such as accented characters into non-accented characters). 

49 

50 This works by decomposing accented characters into the letter and the accent. 

51 

52 The subsequent ASCII encoding drops any accents and leaves the original letter. 

53 

54 Returns the original string if no transliteration is available. 

55 

56 Args: 

57 text (str): a string to be transliterated 

58 

59 Returns: 

60 text (str): the transliterated string 

61 """ 

62 t11e_text = unicodedata.normalize('NFKD', text).encode('ascii', 'ignore').decode('ascii')
